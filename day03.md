# Day 03 

## steps for start django project

1. first step create project diretory.
```
mkdir project1
cd project1
```

2. after create virtual environment & activate it.
```
python -m venv env1
dir
env1\Scripts\activate
```

3. install django in that environment
```
pip install django==5.0
python -m django --version
```

4. create django project & enter vscode IDE througth cmd
```
django-admin startproject mysite
code .
```
